This calculator was made as one of my first projects. I started with buttons but then moved on to text feilds. I also used Js, which is a little more complicated, to force the input to only have 1 sign in the operator text input, ex: you physically canot type out // or ++. The answers are given in Html alerts. There is a different alert when you try to divide by 0, it wont work.

This project is extremely minimalistic, feel free to use css to change the font, colours, and animations!

I made this number guessing game as it is a very simplpe begginer html project. I used a lot of js as it pairs well with html with the use of <script></script>. In this game you get 5 chances to guess the correct number as shown in the in-game description. There are a lot of changeable elements like the amount of guesses that are allowed (will take some copy pasting of the js functions), the biggest and smallest nnumbers (1 - 100 is the default but you can make it go in the negatives as well as how high you want, but be careful as this might make the game way too hard).

This project has a cyber theme to it: green text, the font is called "monospace" but there are many more fonts that could also fit this style that are probably better. The black background also gives it a hacker vibe.
